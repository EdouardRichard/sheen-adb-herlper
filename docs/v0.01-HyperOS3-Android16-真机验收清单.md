# v0.01 HyperOS 3 / Android 16 真机验收清单

> 当前状态：待真机执行。不得以自动化构建替代以下协议、系统 Intent、SAF、ROM 输出和生命周期验收。

## 准备

- 主控端：HyperOS 3 / Android 16；另准备 Android 11+ 非小米设备。
- 被控端开启开发者选项与无线调试；记录只留在本地验收表，不提交真实 IP、端口、序列号、日志或配对码。
- 安装 Release 候选包，确认权限页仅显示网络访问相关普通能力，无相机、定位、存储、通知、无障碍或前台服务。

## 远端直接连接与首次配对

1. 输入无线调试主页面的调试地址，确认状态依次为“连接中 → 已连接”，档案自动建立。
2. 新设备认证失败时确认仅此错误显示“使用配对码”；不可达与超时不得显示该入口。
3. 输入配对地址和 6 位配对码；分别验证成功、错误码、取消、离开页面，确认输入立即清空且无系统/应用日志泄漏。
4. 配对成功后输入调试端口连接；重启应用后验证已配对重连。
5. 旧式 `:5555`（有设备时）验证 RSA 确认期间显示“等待设备授权”，确认/拒绝/超时均可恢复。

## 本机连接

1. 在主控端手动开启无线调试并完成必要配对。
2. 抽屉点击“本机连接”，确认只预填 `127.0.0.1:`，手动填当前调试端口。
3. 连接后在 Shell 执行无害 `echo`，核对输出并主动断开；重启或端口变化后核对诊断建议。

## 页面与会话

1. 未连接时概览、Shell、进程、Logcat 菜单可见、置灰且读屏提示“请先连接设备”。
2. 手机竖屏验证抽屉；平板/横屏验证常驻侧栏双栏布局。以 1.0×、1.3×、2.0× 字体确认无关键控件不可达。
3. 切换档案连接时确认旧会话先关闭；旧设备 Shell/Logcat 结果不得出现在新设备页面。

## 设备概览

1. 核对品牌、制造商、型号、代号、Android/SDK/构建/补丁、ABI/核心、内存、存储、电池、温度、运行时间、网络地址。
2. ROM 不提供字段必须显示“设备未提供”。
3. 留在页面至少 15 秒确认电池/内存约每 5 秒刷新；离开页面、断开、切换设备后通过脱敏诊断确认停止。

## Shell

1. 首次进入显示风险说明；多行命令原样执行，复制内容正确。
2. 验证 stdout/stderr；不支持 shell-v2 时必须显示“合并输出”。
3. 验证 30 秒默认超时、调整到 5 分钟、运行中取消、远端断开与非零退出码状态。
4. 产生超过 1 MiB 输出，确认丢弃最早内容并提示；切换设备/杀进程后历史消失。

## 进程

1. 刷新并搜索进程名/PID，复制 PID 与名称。
2. 核对 PID、UID、状态、RSS；缺失列显示降级说明。
3. 确认无 kill、强停、卸载、冻结或应用管理入口。

## Logcat 与导出

1. 未点击开始时不得采集；默认 main/system/crash、Info、线程时间。
2. 验证 Verbose–Fatal、main/system/crash/radio/events、关键字、暂停/继续、清屏和复制可见内容。
3. 达到 10,000 行或 4 MiB 时确认丢弃最早内容并提示。
4. 点击导出，使用系统文件创建器写入 `.txt`，核对 UTF-8；取消选择不得产生文件。
5. 点击停止、返回页面、断开与切换设备时确认立即停止，无后台通知/服务、无继续写入。

## 设置、删除与系统页面

1. 核对版本、许可证、隐私承诺、风险和支持范围。
2. 验证无线调试/开发者选项 Intent；ROM 不支持时显示手动路径且不自动改设置。
3. 删除单档案需二次确认；最后一个身份引用删除后旧身份不可继续重连。
4. “清除所有本地数据”二次确认后，档案为空、旧身份无法重连、偏好与临时文件不可恢复；重新连接需按系统状态重新配对。

## 结束

- 主动断开，杀进程后确认无后台采集与通知。
- 仅记录“通过/失败、系统版本、脱敏技术代码”；失败材料通过私密渠道处理。
