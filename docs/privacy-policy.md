# Sheen ADB 助手隐私政策

生效日期：2026-07-23

Sheen ADB 助手是纯本地工具，不提供账号、云端服务、广告、统计、遥测、崩溃上报、推送或支付。

## 本地处理的数据

- 用户手动输入的目标主机与端口：用于建立本机或局域网 ADB 连接，并作为设备档案元数据保存在应用私有 DataStore 中。
- ADB 主机身份：私钥以 Android Keystore 中不可导出的 AES-GCM 密钥包装后存放于不可备份的应用私有目录。
- 无线发现结果：只在用户打开局域网发现或本机配对窗口时，通过系统 NSD 处理设备主动公布的 ADB TLS 服务；服务名、真实端点和网络身份只在当前进程内使用，不写入设备档案、日志或磁盘。应用不扫描子网或未公布端口。
- 配对材料：六位配对码、QR service instance/password/payload、临时二维码和通知 token 只在当前 attempt/window 内存中使用；成功、失败、取消、过期、离页或 Session 变化后清除，不写入日志、SavedState、剪贴板或磁盘。
- 本机配对通知：用户进入本机模式并切换到系统设置时，最长 2 分钟显示隐私化通知。锁屏通知没有输入动作；解锁后才可输入。通知标题、正文和公开版本不包含配对码、真实端点或设备身份。
- 设备概览与进程快照：用于当前页面展示，不持久化、不上传。
- 被控端当前用户的第三方应用包名、UID、应用名、图标、启用状态和操作结果：仅用于当前应用管理/进程关联页面与用户明确确认的单包操作。基础 APK 与图标在有界内存中按需读取和解析，不缓存到磁盘；断开、取消或切换 ADB Session 后清除。
- Shell 命令、输出与 Logcat：默认只保存在当前进程的有界内存；结构化 Logcat 与进程/应用关联只用于当前前台分析。切换设备、离开诊断页面或退出应用后清除。仅用户主动复制或通过系统文件创建器导出当前可见内容时离开应用内存。
- 脱敏诊断事件：最多 100 条，仅当前进程内保存，不含真实地址、命令、输出、配对码或密钥。

## 网络与文件

应用的网络能力只用于用户主动连接指定 ADB 地址，或在当前网络发现系统已公布的 ADB TLS 服务；不访问产品后端、不枚举子网、不探测未公布主机/端口。Logcat 导出使用 Android Storage Access Framework，由用户选择目标文件；应用不申请所有文件访问权限，也不自动保存日志。

## 删除

“设置与隐私 → 清除所有本地数据”会删除设备档案、ADB 主机身份、偏好和临时文件，并在操作后检查档案与旧身份不可继续使用。卸载应用也会移除应用私有数据；身份文件禁止系统备份。

## 权限

当前 Manifest 声明 `INTERNET`、`ACCESS_NETWORK_STATE`、`CHANGE_WIFI_MULTICAST_STATE`、`POST_NOTIFICATIONS` 与 `FOREGROUND_SERVICE`。通知权限仅在用户使用本机通知配对时按需请求，拒绝后使用应用内输入；其余四项为安装时普通权限。应用不申请相机、位置、Nearby、存储、蓝牙、通讯录、短信、电话、无障碍、自启动或忽略电池优化权限。完整说明见 `docs/权限矩阵.md`。

## 联系与变更

安全问题请按 `SECURITY.md` 使用私密渠道报告，禁止公开发送敏感设备材料。政策变化会在仓库修订记录和新版本中说明，不会以远程配置静默扩大数据范围。
