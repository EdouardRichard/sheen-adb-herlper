# 第三方依赖与许可证

> v0.01，2026-07-14。版本以 `gradle/libs.versions.toml` 与 Gradle 解析结果为准。所有直接新增依赖均在 Apache-2.0、MIT、BSD-2-Clause、BSD-3-Clause 白名单内。

| 依赖 | 版本 | 许可证 | 用途 | 体积/替代方案 |
|---|---:|---|---|---|
| Android Gradle Plugin | 9.1.1 | Apache-2.0 | Android 构建与 R8 | 仅构建时；无运行时体积 |
| Kotlin / Compose compiler | 2.3.20 | Apache-2.0 | Kotlin 与 Compose 编译 | 技术基线，无等价小型替代 |
| Compose BOM / UI / Material 3 | 2026.06.01 | Apache-2.0 | 简体中文、自适应 UI | 官方按需组件；未引入第三方 UI 框架 |
| Activity Compose | 1.13.0 | Apache-2.0 | Activity、SAF Activity Result | 官方最小集成 |
| AndroidX Lifecycle | 2.10.0 | Apache-2.0 | ViewModel、生命周期安全 StateFlow 收集 | 官方基线；避免自建生命周期层 |
| AndroidX DataStore Preferences | 1.2.0 | Apache-2.0 | 设备档案非敏感元数据 | 小型持久化层；替代为 SQLite 会更重 |
| Kotlin Coroutines | 1.10.2 | Apache-2.0 | 取消、超时、Flow、IO 调度 | 核心异步基线 |
| Kadb | 2.1.1 | Apache-2.0 | ADB、TLS、配对码、Shell 流 | AAR 约 226 KiB，密码学传递依赖较大；适配器可替换为 AOSP/自有实现 |
| Okio | 3.17.0 | Apache-2.0 | Kadb 流与旧式 ADB 合并输出的有界读取 | Kadb 已传递引入，本项目显式固定同版本以适配流；替换随 Kadb 适配器进行 |
| Bouncy Castle bcprov/bcpkix | 1.83 | MIT | 证书与密码学 | 主要体积来源之一；由 R8 裁剪，替换需协议复验 |
| spake2-java | 1.0.5 | Apache-2.0 | Android 11+ SPAKE2 配对 | Kadb 传递依赖，JitPack 限定单一 group |
| ed25519/curve25519-elisabeth | 0.1.0 | MIT | SPAKE2 曲线运算 | 二级传递依赖 |
| AndroidX DocumentFile | 1.1.0 | Apache-2.0 | Kadb Android artifact 传递能力 | 本应用不调用文件管理；若 Kadb 拆包可移除 |
| HiddenApiBypass | 6.1 | Apache-2.0 | Kadb Android TLS 兼容传递能力 | 本应用不直接调用；商店/Android 16 待真机复核 |
| TestNG | 7.11.0 | Apache-2.0 | JVM 单元测试 | 仅测试，不进入 APK |

未引入 DI、导航、图片加载、网络业务、分析、广告、崩溃上报、支付或 JUnit EPL 依赖。Compose 仪器化测试所常用的 JUnit 4 依赖不在许可证白名单，因此 v0.01 使用 TestNG 覆盖 UI 状态/菜单策略与纯逻辑，真机清单覆盖实际 Compose 语义与交互。

依赖源码与许可证来源详见原项目发布页；Kadb 的详细选型和回滚策略见 ADR 0001。
